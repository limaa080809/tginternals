# help.promoDataEmpty
No PSA/MTProxy info is available

```
help.promoDataEmpty#98f6ac75 expires:int = help.PromoData;
```

## Parameters
| Name | Type | Description |
| ---- | :----: | ----------- |
| expires | int | Re-fetch PSA/MTProxy info after the specified number of seconds |


## Type
help.PromoData
