# auth.loginToken
Login token (for QR code login)

```
auth.loginToken#629f1980 expires:int token:bytes = auth.LoginToken;
```

## Parameters
| Name | Type | Description |
| ---- | :----: | ----------- |
| expires | int | Expiration date of QR code |
| token | bytes | Token to render in QR code |
